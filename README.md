## Description
HERO.IO is a modern app discovery and management platform where users can explore, install, and manage their favorite productivity apps. Built with React and Vite, it offers a seamless experience for browsing apps, viewing details, and tracking installations.

## Features
- 🏠 Home page with trending apps
- 📱 All apps page with live search and sort
- 📊 App details with ratings chart
- ⚡ Install & uninstall with localStorage
- 🔍 Loading animations
- 📲 Fully responsive design
- ❌ Custom 404 error page

## Technologies Used
- React.js
- React Router DOM
- Recharts
- React Hot Toast
- React Icons
- Vite
- CSS (Inline + Global)
- LocalStorage API
